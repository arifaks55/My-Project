Soru 1 Cevabı

Bu etkinin arkasındaki mantık, JavaScript kullanarak CSS'in display özelliğini değiştirmektir. Örnekte, JavaScript, bir düğmeye tıkladığınızda (genellikle mobil cihazlarda menüyü açmak için kullanılır) belirli bir HTML öğesinin görünürlüğünü değiştirir. Örnekte, JavaScript, "navbar" adlı bir id'ye sahip bir HTML öğesini alır. Bu, menünün kendisidir. Daha sonra, her tıklamada bu menünün CSS'inin "display" özelliğini kontrol eder. Eğer görünürse, "none" olarak değiştirir ve görünmez hale getirir. Eğer görünmezse, "block" olarak değiştirir ve görünür hale getirir. Bu, menüyü açmak ve kapatmak için bir toggle (açma/kapatma) mekanizması sağlar.

Soru 2 Cevabı

function toggleMenu() {
var x = document.getElementsByClassName("navbar")[0];
if (x.className === "navbar") {
x.className += " responsive";
}
else { x.className = "navbar";
}
}

Bu kodlar, ekran genişliği 600 pikselden küçük olduğunda (tipik bir mobil ekran genişliği), navbarı gizleyip menüyü açma/kapama ikonunu gösterir. Kullanıcı menüyü açtığında, JavaScript bu durumu algılar ve navbarın sınıfına "responsive" sınıfını ekler. Bu "responsive" sınıfı, CSS'deki media query'leri etkinleştirir ve mobil uyumlu görünümü sağlar.
